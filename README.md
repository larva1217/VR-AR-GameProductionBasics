# 🎮 VR/AR/게임제작기초

## 📚 교과목 개요

Unity3D를 이용한 게임제작 기초를 목표로 합니다.

Unity3D의 구성 요소, 기본 기능, 게임에 대한 기초 이론 및 제작 과정 등에 대해 학습합니다.

매주 각 Chapter의 연습 과제 프로그래밍과 기말 프로젝트 제출이 필수입니다.

본 교과목은 산학연계 교과목으로, (주)오르카소프트가 제시한 기말 프로젝트를 구현하기 위해 이론 및 실습 수업을 진행합니다.

학기말에는 제출한 기말 프로젝트를 (주)오르카소프트로부터 평가받습니다.
